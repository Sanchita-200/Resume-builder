# Resume Builder

A modern, client-side Resume Builder web application built with Next.js App Router, TypeScript, and Tailwind CSS.

## Features

- **Multi-Step Builder**: Form sections for Personal Info, Education, Experience, Skills, Projects, Certifications, and Languages.
- **Real-Time Live Preview**: Instant resume rendering as you type.
- **5 Professional Templates**: Modern, Classic, Minimalist, Tech, and Creative layouts.
- **Custom Color Themes**: Indigo, Emerald, Crimson, Amber, Slate, and Violet.
- **Direct PDF Download**: Export high-resolution A4 PDFs locally.
- **Auto-Save**: Form state automatically saved to browser Local Storage.
- **Dark / Light Mode**: Theme toggle for comfortable editing.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
